# CashTrackr AI

CashTrackr AI is a full-stack personal finance web application that helps users add, manage, and categorize expenses through a clean dashboard. It includes authentication, dark/light mode, and AI-powered categorization to automatically organize expense items.

This was my first large-scale project using React, TypeScript, TailwindCSS, Node.js/Express, PostgreSQL, Prisma, and AI integration.

## ✨ Features

- Add and manage expenses
- View expenses on a dashboard
- AI-powered expense categorization
- Login and register pages
- Dark and light mode
- Clean, responsive UI based on a custom Figma design
- Video walkthrough showcasing the full project

## 🛠 Tech Stack

### Frontend
- React with Vite
- TypeScript
- TailwindCSS
- React Router

### Backend
- Node.js and Express
- PostgreSQL
- Prisma ORM
- JWT authentication
- bcrypt password hashing

### AI Integration
- Google Generative AI SDK

### Development Tools
- DBeaver
- ESLint
- Nodemon

## 📌 Project Status

Core features are complete, including authentication, expense management, dashboard display, theme switching, and AI categorization.
